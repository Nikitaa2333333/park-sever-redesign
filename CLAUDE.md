# CLAUDE.md — Парк Север

Редизайн сайта **park-sever.ru** — загородный парк со стеклянными гостевыми домами (дер. Василёво, Дмитровский г.о., ~80 км от Москвы). Стек — **Astro (SSG)**, многостраничный сайт.

## Текущий фокус

Быстро и качественно перерабатываем **главную** по принципам дизайна (ниже). Работаем **быстро, без лишних церемоний** — минимум процесса, максимум результата на экране. Старые планы (аудит, роадмап, дизайн-канон `verstka/`) убраны в `_archive/`.

## Контент: JSON отложен (вернёмся позже)

Раньше правило было «источник правды — `content/*.json`». **Сейчас оно снято для переработки главной:** верстаем главную напрямую в разметке (`src/pages/index.astro`), контент пишем прямо в компоненте, не подгоняя под JSON-модель.

**Как отвязываемся (важно — не дрейфовать назад к JSON):**
- `content/*.json` и `content/SCHEMA.md` — **НЕ источник правил/структуры**, только пассивный склад дословных текстов. Ничего под JSON не подгоняем.
- По мере переработки блока — переносим его контент **инлайном** в `index.astro` и перестаём читать эту секцию из JSON.
- Когда `index.astro` перестанет импортировать `content/` — переносим папку (и `SCHEMA.md`) в `_archive/`. Пока не двигаем: от неё ещё зависит сборка.

## Дизайн (как на главной сейчас)

**Шрифты (2):**
- **Monotype Corsiva** — заголовки и акценты (заказной курсивный скрипт). Локально, `--font-display`.
- **Lora** — весь наборный текст и UI (кнопки, меню, eyebrow, числа). `--font-head` / `--font-body`.

**Цвет:**
- Поверхности: холст `--canvas #fff`, парчмент `--cream #f3f1eb`, тёмно-коричневая сцена `--bark #2E2218`, песочная `--sand`, зелёная `--green`.
- Текст: `--ink #28262a` (почти чёрный).
- Акценты: **преобладает синий `--navy #0F2A47`** (структура: кнопки, ссылки); металл — золото `--gold #C9A86A`, медь `--copper`.

**Принципы:**
- **Текст — всегда сплошной цвет.** На светлом фоне — чёрный (`--ink`), на тёмном — **чисто белый `#fff`**. Никакой серости/прозрачности. CSS `opacity` для приглушения текста и цвета **запрещён** — приглушение делаем сплошным цветом.
- Кнопки — **прямые углы** (`--r-btn:0`). **Теней нет** — глубина строится контрастом поверхностей и тонкой линией.
- Много воздуха. Контейнер `--wrap:1200px`.

## Ключевые файлы

- `content/*.json` — контент (источник правды); `content/SCHEMA.md` — формат.
- `src/lib/content.ts` — загрузка JSON + резолв `ref`.
- `src/pages/index.astro` — главная.
- `src/styles/design-system.css` — дизайн-система (токены + классы).
- `src/layouts/SiteLayout.astro` — каркас, шрифты, инициализация виджета брони.

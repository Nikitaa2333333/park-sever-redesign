# Дизайн-система «Север» — единый источник правды по визуалу

> Один файл отвечает за весь визуал: токены, типографика, ритм секций, поверхности, компоненты, анимация.
> **Контент сюда не входит** — он живёт в `content/*.json` (см. корневой `CLAUDE.md`).
> Реализация — `src/styles/design-system.css`. Этот MD = спека, CSS = её воплощение. Расходятся — чини CSS под MD.
> Если в коде встречаешь паттерн, противоречащий этому файлу, — правь код, **не выдумывай новое правило**. Сомневаешься — спроси человека.

**Канон обновлён (2026-06-23) под прямые принципы заказчика (`Главная.docx`).** Регистр — тёплый редакционный премиум. Два семейства акцентов: **navy** (структурная глубина) + **тёплый металл — золото/медь** (заголовки, номера, нарядность). Поверхности — не только white/cream/ink: добавлены тёплые сцены **bark / green / copper / navy**. **Кнопки — прямые углы** (капсулы отменены). **2 шрифта:** заголовки — **Monotype Corsiva** (заказной каллиграфический, курсив), наборный текст и весь UI — **Lora (serif)**. См. «Что изменилось» в конце.

---

## 1. Философия

Тёплый редакционный премиум. **Воздух важнее декора**, но регистр — не холодный: тепло задаётся золотом/медью в заголовках и номерах, тёмными «обжитыми» сценами (bark) и природными фонами (green). Глубина создаётся **контрастом поверхностей** (white ↔ cream ↔ bark/green/navy) + волосяной/тонкой светлой линией там, где нужна граница карточки. Ритм — пробелами и сменой фона секций по смыслу блока.

Каждый элемент берёт **именованный класс** из этого файла. Ad-hoc inline-стили (кроме точечных отступов) — нежелательны: если повторяется — это новый класс, и он описывается здесь.

## 3. Типографика — закрытая шкала

Заголовки — **Monotype Corsiva** (каллиграфический курсив), наборный текст — **Lora (serif)**. Каждый элемент = один класс.

### Заголовки (`--font-display`, Monotype Corsiva)
| Класс | Спека | Где |
|---|---|---|
| `.hero-title` | display, `clamp(44px,7vw,84px)`, lh 1.05 | H1 (hero), «Парк Север» — крупнее остального |
| `.h1` | display, `clamp(38px,6vw,64px)`, lh 1.0 | крупный заголовок |
| `.h2` | display, `clamp(30px,3.6vw,46px)`, lh 1.06 | опенер секции |
| `.h3` | display, `clamp(19px,2vw,24px)`, lh 1.2 | подзаголовок |

⚠️ Каллиграфический скрипт — **вес обычный (500), `letter-spacing:0`** (фейк-болд и отрицательный трекинг ломают связные глифы).

Цвет заголовка: на светлом — `--ink`; на тёмных сценах (`.sec--navy/--ink/--bark`) — белый (автоматически). **Золотой заголовок** — модификатор `.h--gold` (`color:var(--gold)`): применяется к нарядным блокам («Искусство отдыха», «Атмосфера для двоих») по требованию заказчика.

### Акцентная строка hero
- **`.hero-accent`** — одна строка-«подпись» в hero (`--font-display`, курсив), **крупнее наборного, но мельче `.hero-title`** (ориентир `clamp(22px,3vw,34px)`). Это намеренное исключение: курсивный display в роли акцента (строка «Вы заслужили несколько дней тишины…»). Размещается над строкой «Деятельность ведётся…».
- Наборный текст hero («Сюда приезжают…») — **уменьшенный межстрочный** (lh ≈ 1.35), Lora.

### Наборный текст и подписи
| Класс | Спека | Где |
|---|---|---|
| `.lead` | Lora 500, 19px, lh 1.55, max `60ch` | лид под заголовком |
| `.body` | Lora 500, 16px, lh 1.6, max `64ch` | основной абзац |
| `.eyebrow` | Lora 600, 14px, **navy** (на тёплых сценах — `--gold`), tracking 0 | надстрочник над H2 |
| `.hl` | navy | выделенное слово **внутри** заголовка |
| `.caption` | Lora 500, 13px, светлый на тёмной плашке | подпись под фото в карточке «Искусство отдыха» |

Размер-уровня **ниже 13px не существует** (подпись `.caption` — минимальный, мелкий по требованию заказчика). Основной текст — не ниже 14px.

### Шапка секции
Группа `eyebrow + H2 + лид` оборачивается в `.shead` (центр) или `.shead--left`. Вертикальный ритм — только `gap`.

## 4. Ритм секций

| Класс | Фон | Когда |
|---|---|---|
| `.sec` | `--canvas` (белый) | дефолт; паддинг `96px 0` (моб. `64px`) |
| `.sec--cream` | `--cream` | вторая светлая поверхность, чередование |
| `.sec--bark` | `--bark`, текст светлый | тёмно-коричневая «обжитая» сцена («Оазис»); текст — на светлой плашке `--paper` |
| `.sec--green` | `--green`, текст светлый | природный фон блоков/карточек |
| `.sec--copper` | `--copper`, текст светлый | тёплый медный фон блоков/карточек |
| `.sec--navy` | `--navy`, текст белый | акцентная глубокая полоса («Атмосфера для двоих», CTA, контакты) |
| `.sec--ink` | `--ink`, текст светлый | редкие кинематографические full-bleed |

Белый/крем — база; тёмные и цветные сцены — **по смыслу блока**, чтобы было тепло и не пусто (заказчик: блоки не должны быть холодными/пустыми). **Не чередовать механически**. На светлых крупных раскладках («Искусство отдыха») пустой фон по бокам — заполнять (тёплая сцена/паттерн), не оставлять голым.

## 5. Layout-паттерны (фактические классы)

- **`.wrap`** — контейнер `max-width 1200px`, паддинг `0 32px`.
- **`.wrap--narrow`** — узкий контейнер (≈`880px`) для блока «Дом» (он инструмент, не центр — заказчик: сделать уже).
- **`.hero`** — full-bleed видео/фото, `min-height 90vh`, тёмный скрим, текст по центру; внизу — строка модуля бронирования (znms) и бейдж «Деятельность ведётся…».
- **`.split2`** — 2 колонки `1fr/1fr`, gap 64px. Модификатор **`.split2--rev`** — фото слева / текст справа (вариант блока «Дом»).
- **`.gallery`** — мозаика, радиус `--r-img`.
- **`.grid3` / `.grid4`** — сетки карточек, gap 24px.
- **`.stats`** — ряд «голых чисел».
- **`.cmap`** — контакты + карта.
- **`.stories-split`** — блок «Истории Парка» пополам: слева лента ВК, справа виджет отзывов Яндекс (раскладка готовится в фазе разметки).

## 6. Поверхности

- **Тени — нет.** Глубина = контраст поверхностей + тонкая линия (`--fog` / `--hair-gold`).
- **Карточка `.card`** — радиус `--r-card` (оставляем), граница `1px --fog`; вариант **`.card--frame`** — тонкая светлая золотистая обводка `--hair-gold` вокруг всей карточки (фото + подпись), по image5. Ховер — лёгкий зум картинки `scale(1.03)`, не тень.
- **Рамка-«картина» для каруселей** — фото в широком паспарту (`.slider-frame`: толстый светлый `--paper`/белый кант + тонкая линия), **без скругления фото**. Это замена прежнему скруглённому слайду (заказчик: «как картина»).
- **Радиусы:** карточки `16`, картинки/карта `24`, **кнопки `0`**, бейдж `30`.
- **«Голые числа»** (`.stats`): число берёт глаз размером и цветом (navy или `--gold`), в карточку не заворачивается.
- **Номер шага 01-06** — поверх фото, **золотом** (`.art-step__num--gold`), не сбоку.

## 9. Анти-паттерны (увидел — убери)

- **Серость/opacity на наборном тексте** — всегда сплошной (`--ink` / светлый).
- **Тень** любая — глубина контрастом и линией.
- **Положительная разрядка** (`tracking-wide`) — только отрицательный tracking на UI/Lora, не на заголовках.
- **`uppercase`** на заголовках/кнопках/eyebrow.
- **Курсив/наклон в наборном тексте и UI** — нет. Исключения: заголовки (`--font-display` намеренно каллиграфический) и **именованная `.hero-accent`**.
- **Основной текст < 14px** (подпись `.caption` — единственное исключение, мелкая).
- **Hex инлайн** вместо токена; `gray-*`; бренд-красный.
- **Иконка/стрелка внутри `.btn`** (стрелки карусели — отдельный контрол, допустимы).
- **Скруглённые («капсульные») кнопки** — углы прямые.
- **Золото/медь как заливка крупных текстовых полей** — это акцент и нарядность, не фон абзацев.
- **Голый пустой фон** по бокам крупных раскладок — заполнять тёплой сценой.

> Прежние запреты «золота нет», «единственный акцент navy», «третья поверхность запрещена», «капсульные кнопки» — **отменены** (см. «Что изменилось»). Не возвращать без указания человека.

## Что изменилось относительно прежнего «закона» (история решения)

**Редакция 2026-06-23 (по `Главная.docx`)** — заказчик сознательно развернул канон 2026-06-21:

1. **Акцент:** было «navy — единственный акцент, золота нет» → стало **navy + тёплый металл (золото `--gold`, медь `--copper`)**: золото для заголовков нарядных блоков и номеров 01-06.
2. **Поверхности:** было «только white/cream/ink, третья поверхность запрещена» → добавлены тёплые сцены **bark (тёмно-коричневый «Оазис»), green, copper**; navy используется как фон «Атмосферы». Текст на тёмном — на светлой плашке `--paper` («как на бумаге»).
3. **Кнопки:** было капсула `--r-pill:100px` → стало **прямые углы `--r-btn:0`**.
4. **Карусель:** скруглённый слайд + круглые стрелки → **паспарту-«картина» без скругления, прямоугольные стрелки, точки-чёрточки**.
5. **Карточки:** добавлен вариант `.card--frame` (тонкая золотистая обводка), подпись на тёмной плашке, смещение второго ряда.
6. **Типографика:** разрешена курсивная `.hero-accent` (display) как именованное исключение.

Сохранено из прежнего канона: 2 шрифта (Corsiva + Lora), тени нет, сплошной наборный текст, радиусы карточек/фото 16/24, волосяная линия для границ. **Если эти решения пересматриваются — менять здесь, затем синхронно в `design-system.css`.**
